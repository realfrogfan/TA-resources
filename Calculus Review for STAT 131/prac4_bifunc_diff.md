Practice Problems: Bivariate Functions & Differentiation
================

1.  Find the domain of the function $f(x, y) = \log(y - x^2)$. Describe
    the region geometrically and sketch it in the $x$-$y$ plane.

2.  Calculate the first-order ($f_x, f_y$) and second-order
    ($f_{xx}, f_{yy}, f_{xy}$) partial derivatives for the following
    functions:

    1)  $f(x, y) = x^4 y^3 - 2x + 5y$
    2)  $f(x, y) = x e^{x^2 + y}$

3.  Let $$
    f(x, y) = \frac{1}{24} xy^2 (3x^2 + y).
    $$ Find the first-order partial derivatives ($f_x, f_y$) and the
    second-order partial derivatives ($f_{xx}, f_{yy}, f_{xy}$).

4.  (More Challenging) Find the first-order ($f_x, f_y$) and
    second-order ($f_{xx}, f_{yy}, f_{xy}$) partial derivatives for the
    following functions:

    1)  $f(x, y) = \log(x^2 + y^2)$
    2)  $f(x, y) = \frac{x + y}{x - y}$
    3)  $f(x, y) = x e^{y/x}$
    4)  $f(x, y) = x \cos(\sqrt{xy})$

5.  Show that the function $u(x, t) = \sin(x - ct)$ satisfies the wave
    equation $u_{tt} = c^2 u_{xx}$.

Practice Problems: Function Basics
================

1.  Consider the function defined by: $$
    f(x) = \begin{cases} 
    \frac{1}{x}, & 2 < x < 5 \\
    0, & \text{otherwise}.
    \end{cases}
    $$

    1)  What is the **Domain** of this function?
    2)  What is the **Support** of this function?

2.  Find the inverse function $x = f^{-1}(y)$ for the function:
    $$f(x) = 3x - 7$$

3.  Explain why the function $f(x) = x^2 - 4$ does not have an inverse
    on the domain $\mathbb{R}$.

4.  Find the domain of the function: $$f(x) = \log(5 - x)$$

5.  Simplify the expression: $$f(x) = e^{3 \log(x)}$$

6.  Plot out the following piecewise function and rewrite it using the
    Indicator Function notation $I_A(x)$: $$
    f(x) = \begin{cases} 
    10, & -2 \le x \le 2 \\
    0, & \text{otherwise}.
    \end{cases}
    $$

7.  Find the inverse function $x = f^{-1}(y)$ for the function:
    $$f(x) = \frac{x}{x + 1}$$

8.  Find the inverse function $x = f^{-1}(y)$ for the function:
    $$f(x) = \log(x - 3)$$

9.  Consider the function: $$
    f(x) = \begin{cases} 
    \sqrt{1 - x^2}, & -1 < x < 1 \\
    0, & \text{otherwise}.
    \end{cases}
    $$

    1)  Determine the domain of $f$.
    2)  Determine the support of $f$.
